Word boundaries regular expression - \b

Matches a zero-width boundary between a word-class character and either a non-word class character or an edge.

\b matches the empty string at the beginning or end of a word.

Examples:

Reg exp Text
\bis\b This island is beautiful!
\bis This is one simple example, isn't it?

White space regular expression - \s

Matches a whitespace character, which in ASCII are tab, line feed, form feed, carriage return, and space; in Unicode, also matches no-break spaces, next line, and the variable-width spaces (amongst others).

Examples:

Reg exp Text
\sis This island is beautiful!
\s This is one simple example ,isn't it?
